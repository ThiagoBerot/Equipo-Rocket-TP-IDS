Para crear el entorno virtual con las dependencias necesarias y correr el contenedor de docker con la base de datos, ejecute sudo bash init.sh.
Para desplegar pagina web ejecute sudo bash  scriptWeb.sh.
Para desplegar la API ejecute: sudo bash scriptApi.sh en otra terminal.
app.py despliega una pagina web en el puerto 5000; api.py despliega una base de datos mysql hecha con docker.
